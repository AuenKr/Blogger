To run locally

1. set all env variable in sample.env and sample.wrangler.toml

With Docker

    - In Linux
    > docker compose up

    - In Windows
    > docker-compose up

Without Docker
> Install node

> For Backend
    Go to backend and run `npm install` and then `npm run dev`

> For Frontend
    Go to frontend and run `npm install` and then `npm run dev`
